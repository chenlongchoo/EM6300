Running the Demo:

The demo is the same as the “Graphics Object Layer Demo”. The only difference is that this version of the demo will only run on the PIC24FJ256DA210 Development Board with the color look up (CLUT) enabled. The images used in this demo are specially processed to share a common palette and the same palette colors are programmed into the PIC24FJ256DA210 color look up table registers.

 

Please refer to the “Graphics Object Layer Demo” for the detailed description of the demo.
