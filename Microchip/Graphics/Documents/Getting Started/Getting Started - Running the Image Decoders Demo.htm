Running the Demo:

This demo shows the image decoding capabilities of “Image Decoders Library”. The demo decodes various bitmap and jpeg images and demonstrates the usage of the image decode API with various options like fitting into an area or cropping into an area, etc. All the images are stored in the internal flash as constant arrays of bytes declared in FlashImageData.c/.h files (FlashImageData.c/h files are generated using the tool Graphics Resource Converter with some manual modifications). The demo does not use Microchip’s MDD file system but provides the required file system APIs to access the image files stored in the ROM through a separate layer (FlashImage.c/.h). 

 

Demo Procedure

Connect the necessary hardware and program with the correct pre-generated hex from the ‘Precompiled Hex’ folder

After programming the demo will loop through different screens showing the effects of different parameters to the decoding API.
